# 感谢页推荐区块说明

## 概述

`extensions/thank-you-referral-block` 是一个 Checkout UI extension，用于在 Shopify 感谢页展示推荐奖励卡片。

## Extension Target

```toml
target = "purchase.thank-you.block.render"
```

商家可以在 checkout editor 中把该 block 添加到感谢页，并调整展示内容。

## 可配置项

配置项定义在 `extensions/thank-you-referral-block/shopify.extension.toml`：

- `image_url`: 左侧图片 URL。
- `title1`: 标题1文案。
- `title2`: 标题2文案。
- `description`: 描述文案。
- `button_label`: 按钮文案。
- `button_url`: 按钮链接。
